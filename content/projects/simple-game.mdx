Simple Game merupakan nama dari aplikasi yang saya kerjakan untuk belajar Next.Js dan Typescript, seperti namanya
ini merupakan sebuah game. yang menerapkan websocket untuk mengirimkan event ke semua pengguna yang berada di area tertentu.

[![Image Project](/image-showcase/SimpleGameShowcase.png)](/image-showcase/SimpleGameShowcase.png)

Tech Stack yang digunakan:

- React Native
- Nest.js
- Mongose
- MongoDb
- Socket.Io

Github client repo klik [disini](https://github.com/marsmallowz/SimpleGame).

Github server repo klik [disini](https://github.com/marsmallowz/simple-game-api).

## Tasks I do

Berikut adalah beberapa tugas atau fitur **(Frontend & Backend)** yang saya kerjakan dalam pengembangan aplikasi:

### Authorization & Authentication

- Membuat Halaman Register.

  Persyaratan:

  - Pengguna dapat mendaftar ke aplikasi menggunakan email, username dan password.
  - Pengguna akan menerima email untuk melakukan verifikasi akun.

[![Image Project](/simple-game-screenshot/sign-up.png)](/simple-game-screenshot/sign-up.png)

- Membuat Halaman Login.

  Persyaratan:

  - Pengguna dapat masuk ke aplikasi menggunakan email atau username dan password.
  - Pengguna akan di arahkan ke homepage jika telah login.

[![Image Project](/simple-game-screenshot/sign-in.png)](/simple-game-screenshot/sign-in.png)

### Homepage

- Membuat Homepage.

  Persyaratan:

  - Pengguna dapat melihat area.
  - Pengguna dapat pindah-pindah subarea.
  - Pengguna dapat melihat detail subarea.

[![Image Project](/simple-game-screenshot/homepage.png)](/simple-game-screenshot/homepage.png)

### SubArea

  - Pengguna dapat melihat detail pengguna.
