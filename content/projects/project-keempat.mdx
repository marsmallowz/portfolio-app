Tukar-Menukar merupakan nama dari aplikasi Skill Swap yang saya kerjakan untuk belajar Next.Js dan Typescript.

[![Image Project](/image-showcase/Tukar-MenukarShowcase.png)](/image-showcase/Tukar-MenukarShowcase.png)

Tech Stack yang digunakan:

- React.js
- Next.js
- Tailwind
- Prisma
- MongoDb

Github repo klik [disini](https://github.com/marsmallowz/tukar-menukar-app).

Live Demo klik [disini](https://tukar-menukar-app.vercel.app/).

## Tasks I do

Berikut adalah beberapa tugas atau fitur **(Frontend & Backend)** yang saya kerjakan dalam pengembangan aplikasi:

### Authorization & Authentication

- Membuat Halaman Register.

  Persyaratan:

  - Pengguna dapat mendaftar ke aplikasi menggunakan email dan username.
  - Pengguna akan menerima email untuk melakukan verifikasi akun.

[![Image Project](/project-keempat-screenshot/auth-register.png)](/project-keempat-screenshot/auth-register.png)

- Membuat Halaman Login.

  Persyaratan:

  - Pengguna dapat masuk ke aplikasi menggunakan email dan password.
  - Pengguna akan di arahkan ke homepage jika telah login.

[![Image Project](/project-keempat-screenshot/auth-login.png)](/project-keempat-screenshot/auth-login.png)

- Membuat Halaman Verify.

  Persyaratan:

  - Pengguna melakukan verifikasi dengan mengatur passwordnya.

[![Image Project](/project-keempat-screenshot/auth-verify.png)](/project-keempat-screenshot/auth-verify.png)

### Homepage

- Membuat Homepage.

  Persyaratan:

  - Pengguna dapat melihat info aplikasi.
  - Pengguna dapat melihat skill-skill pupuler.
  - Pengguna dapat melihat fitur-fitur aplikasi.

[![Image Project](/project-keempat-screenshot/homepage-2.png)](/project-keempat-screenshot/homepage-2.png)

### Dashboard

- Membuat Dashboard.

  Persyaratan:

  - Pengguna dapat melihat daftar exchange yang dimiliki pengguna.
  - Pengguna dapat mengklik exchange yang ada untuk menuju halaman detail exchange.
  - Pengguna dapat menuju halaman list pengguna dengan mengklik tombol create exchange.

[![Image Project](/project-keempat-screenshot/exchange-list.png)](/project-keempat-screenshot/exchange-list.png)

### Users

- Membuat Halaman Daftar Pengguna.

  Persyaratan:

  - Pengguna dapat mencari pengguna lainnya.
  - Pengguna dapat melihat daftar pengguna lainnya.
  - Pengguna dapat menuju halam detail pengguna lainnya, dengan mengklik pengguna lainnya yang tersedia.

[![Image Project](/project-keempat-screenshot/user-list.png)](/project-keempat-screenshot/user-list.png)

- Membuat Halaman Detail Pengguna.

  Persyaratan:

  - Pengguna dapat melihat detail pengguna dan pengguna lainnya.
  - Pengguna dapat membuat exchange dengan mengklik tombol create exchange.

[![Image Project](/project-keempat-screenshot/user-details.png)](/project-keempat-screenshot/user-details.png)

### Settings

- Membuat Halaman Update Profile.

  Persyaratan:

  - Pengguna dapat memperbaruhi profilenya seperti username maupun foto profile.

[![Image Project](/project-keempat-screenshot/settings.png)](/project-keempat-screenshot/settings.png)

- Membuat Halaman Mengatur Skill.

  Persyaratan:

  - Pengguna dapat mencari skill.
  - Pengguna dapat melihat daftar skill.
  - Pengguna dapat mengatur skill apa yang ingin pengguna inginkan dan bagikan.

[![Image Project](/project-keempat-screenshot/skill-list.png)](/project-keempat-screenshot/skill-list.png)
