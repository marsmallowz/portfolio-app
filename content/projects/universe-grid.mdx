Universe Grid merupakan nama dari game puzzle yang saya kembangkan,
game ini menerapkan aturan Conway's Game of Life yang
merupakan Automaton Cellular sebagai dasarannya.

### Memiliki beberapa mode yang dapat dimainkan :

- Mode yang menerapkan aturan Conway's Game of Life :
  - Mode Survival
  - Mode Destroying
  - Mode Pattern
  - Mode Pattern X
- Mode yang menerapkan aturan Conway's Game of Life + aturan Universe Grid :
  - Mode X (Ongoing / Under Development)

[![Image Project](/image-showcase/UniverseGrid.png)](/image-showcase/UniverseGrid.png)

### Tech Stack yang digunakan:

- Godot Engine
- Admob
- Google Play Console
- Google Play Game Service

Untuk lebih jelas dapat langsung akses melalui :

- Google Play Store klik disini (Ongoing)
