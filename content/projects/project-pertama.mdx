Instagram Clone merupakan aplikasi yang saya kerjakan sebagai **Project Latihan** di Purwadhika.

Tech Stack yang digunakan:

- React.js
- Express.js
- Tailwind
- Sequelize
- Mysql

## Tasks I do

Berikut adalah beberapa tugas atau fitur **(Frontend & Backend)** yang saya kerjakan dalam pengembangan aplikasi:

### Authorization & Authentication

- Membuat Halaman Register.

  Persyaratan:

  - Pengguna dapat mendaftar ke aplikasi menggunakan email, phonenumber, username, fullname dan password.
  - Pengguna akan menerima email untuk melakukan verifikasi akun.

[![Image Project](/project-pertama-screenshot/auth-register.png)](/project-pertama-screenshot/auth-register.png)

- Membuat Halaman Login.

  Persyaratan:

  - Pengguna dapat masuk ke aplikasi menggunakan email atau phonenumber atau username dan password.
  - Pengguna akan di arahkan ke homepage jika telah login.

[![Image Project](/project-pertama-screenshot/auth-login.png)](/project-pertama-screenshot/auth-login.png)

### Homepage

- Membuat Homepage.

  Persyaratan:

  - Pengguna harus login untuk dapat mengakses homepage.
  - Pengguna yang belum login akan diarahkan ke halaman login.
  - Pengguna dapat menggunakan melihat postingan dari pengguna lain.
  - Pengguna dapat memberikan comment dan like pada sebuah postingan.

[![Image Project](/project-pertama-screenshot/homepagex.png)](/project-pertama-screenshot/homepagex.png)

### Post

- Membuat Fungsi Create Post.

  Persyaratan:

  - Pengguna dapat menambahkan postingan berupa foto beserta keterangan.

[![Image Project](/project-pertama-screenshot/post-create.png)](/project-pertama-screenshot/post-create.png)

### Profile

- Membuat Halaman Profile (Data Statik).

  Persyaratan:

  - Pengguna dapat melihat proile pengguna sendiri atau pengguna lainnya.

[![Image Project](/project-pertama-screenshot/profile.png)](/project-pertama-screenshot/profile.png)
